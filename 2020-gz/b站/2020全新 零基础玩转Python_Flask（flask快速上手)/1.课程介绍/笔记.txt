使用虚拟环境
步骤:
1.创建虚拟环境
pip install virtualenv

windows---->设置环境变量---->
(如果有多个python版本,可以通过命令:virtualenv -p pytnon路径 环境名,来创建虚拟环境)
2.进入环境下得Scripts文件夹下
输出命令执行脚本:activate

3.接下来就可以在该虚拟环境下pip安装包或者做各种事了
pip install django==2.0

退出虚拟环境

输入命令:deactivate
